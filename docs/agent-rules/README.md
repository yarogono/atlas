# 🤖 Agent & Code Conventions (AI 에이전트 규약)

이 폴더는 AI 에이전트와 협업할 때 지켜야 할 **코드 규약(Coding Conventions)**, **아키텍처 구조(Architecture Guidelines)**, 그리고 **프롬프트 가이드라인**을 정리해 두는 곳입니다.

나중에 아래와 같은 주제로 마크다운(`.md`) 파일들을 추가하여 정리해 보세요:

- `project-structure.md`: 프로젝트 폴더 구조 및 컴포넌트 분리 원칙
- `naming-conventions.md`: 변수, 함수, 파일 네이밍 규칙
- `git-workflow.md`: 브랜치 전략 및 커밋 메시지 규칙
- `tech-stack.md`: 사용 중인 기술 스택 및 버전 정보

## 💡 작성 팁
AI 에이전트에게 작업을 지시할 때 "이 폴더의 `naming-conventions.md` 규칙을 반드시 지켜줘"라고 명시하면, 에이전트가 항상 일관된 스타일로 코드를 작성하게 됩니다.
