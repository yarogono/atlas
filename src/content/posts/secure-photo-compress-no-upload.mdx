## 😱 우리가 미처 인지하지 못했던 '서버 업로드'의 은밀한 위험성

이미지 용량을 줄이기 위해 대다수가 이용하는 사이트들은 구조적으로 사용자가 등록한 이미지를 **'그들의 서버(Server)'로 먼저 전송(Upload)**합니다. 

문제는 우리가 올리는 사진에 다음과 같은 중요하고 기밀성이 높은 개인정보가 가득 담겨 있다는 점입니다:
* **개인 신분증 및 계약 증빙**: 주민등록증, 운전면허증 사본, 통장 사본, 부동산 계약서 등 세무/행정 목적으로 촬영한 초고해상도 사진.
* **이력서 증명사진 및 가족 일상**: 얼굴 데이터가 고스란히 담긴 생체 정보.
* **기업 보안 문서 캡처**: 사내 시스템 대시보드 스크린샷, 보고서 파일 등 외부 공개가 금지된 기밀 자료.

아무리 해당 서비스들이 "업로드 후 몇 시간 이내에 자동 파기한다"고 정책상 기술해 두었을지라도, 전송 과정에서의 패킷 탈취나 서버 저장소 해킹, 내부 직원의 유출 가능성 등 정보 유출 리스크로부터 안전하지 못합니다.

## 🔒 인터넷 연결을 끊어도 동작하는 100% 로컬 브라우저 연산 원리

본 안심 포털의 **[무료 이미지 압축기](/compress)**는 기존 웹 압축 기술과 개념부터 완전히 다릅니다. 사용자가 파일을 드롭존에 올려도 외부 네트워크로 사진 데이터를 송출하지 않습니다.

### 1. Canvas API 기반의 순수 프론트엔드 연산
자바스크립트의 표준 Canvas API를 브라우저 내에서 구동하여 이미지를 복제하고 지정된 압축 품질(Quality)과 가로/세로 비율로 픽셀을 새로 드로잉합니다. 즉, 모든 인코딩과 디코딩 연산이 사용자의 **핸드폰 혹은 컴퓨터의 RAM(메모리) 상에서만** 실행되고 다운로드와 동시에 휘발됩니다.

### 2. 인터넷 선을 뽑고 테스트해 보세요
본 서비스가 100% 로컬에서만 도는 진정한 보안 툴킷인지 의심되신다면, **네트워크 연결(Wi-Fi, 모바일 데이터)을 완전히 차단한 상태**로 사진 압축을 돌려보세요. 외부 서버 통신이 전혀 없기 때문에 오프라인 상태에서도 리사이징과 WebP 변환, 품질 압축이 정상적으로 단숨에 처리되는 놀라운 안전성을 직접 두 눈으로 검증하실 수 있습니다.

### 3. 광고 없는 쾌적함과 똑똑한 ZIP 일괄 다운로드
단 한 건의 강제 전면 광고도 노출하지 않아 쾌적함을 선물하며, 압축에 성공한 여러 장의 사진들을 하나로 패키징하는 `JSZip` 엔진을 통해 사용자의 파일 다운로드 차단 경고를 우려한 **원터치 ZIP 일괄 아카이빙** 기능까지 완벽하게 호환됩니다.

## 🛠️ 안전한 이미지 파일 압축 실행 3단계

1. **[무료 안심 사진 압축기](/compress)**를 실행합니다.
2. 기기에서 보안 처리가 필요하거나 크기를 다이어트하려는 이미지들을 다중 선택하여 올립니다.
3. 원하는 해상도 크기와 포맷(WebP, JPEG, PNG)을 설정합니다. (왜곡을 완벽히 피하면서 커스텀 픽셀을 입력하고 싶다면 **크기 조정 '직접 지정'**을 탭하고 가로, 세로 입력 후 '비율 고정 유지'를 활성화합니다.)
4. **'한 번에 압축하기'**를 탭해 연산을 즉각 완료한 후, 디테일 분리 슬라이더를 통해 화질 유지 상태를 돋보기로 보듯 비교 분석합니다.
5. 안전하게 생성된 **'전체 압축파일(ZIP) 다운로드'**를 통해 파일 앱으로 원터치 저장합니다.

정보 보안이 생명인 현대 사회에서 무의식적으로 행했던 이미지 서버 업로드 습관은 매우 위험합니다. 소중한 내 개인 데이터와 회사 기밀을 완벽하게 수호하는 무설치 무료 로컬 압축기를 북마크에 지정하고 털릴 걱정 없이 편리하게 사용하세요!
